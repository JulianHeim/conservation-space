Sirma Platform : Edit Comment

  1. The creator of comment may edit their own comments.
  2. Any user may edit the body text of his/her own reply on comments. Comments associated with certain 1st class objects (e.g., projects, tasks or objects) that have been locked or are in a certain predefined states may not be edited without special override permissions. (Not Ready)
  3. The system admin and the project manager may edit any and all comments.

Any comment's metadata (except for the create date/time and creator fields), body text, and replies may be edited after being saved.
The comment status could be changed from Edit comment functionality.

  • Edit Type
  • Edit Status
  • Edit Body Text

Edit Type

  • The user opens the comment and clicks on the link of the comment type (1-2).

  • The system opens screen where the user selects another comment type from a drop-down menu and saves the changes (1-2-3).

Edit Status

  • The user opens the comment and clicks on the link of the comment status (1-2).

  • The system opens screen where the user selects another comment status from a drop-down menu and saves the changes (1-2-3).

Edit Body Text

  • The user opens the comment and clicks on "Edit Details"  from the drop-down menu of the opened body text (1-2-3-4).

  • The system opens the comment's body text, the user edits it and saves the changes (1-2).

  • The changes are saved in the comment"s body.
